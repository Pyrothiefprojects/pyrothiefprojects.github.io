Project Nova is a multi-genre game framework built from custom engines, each powering a Nova title and serving as a reusable dev kit. It comprises four interconnected stories, all set in the Nova Universe.

The four titles:

  • Spaceship combat roguelike
  • Browser-based point-and-click adventure
  • Squadron-based space dogfighter
  • Survival exploration

The goal is to ship all four games as finished products while also releasing the engines behind them — giving players and creators the tools to build their own content in the genres each engine specializes in.
